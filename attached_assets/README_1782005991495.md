# Handoff: WKRLY Group — Homepage Redesign (Deep Navy)

## Overview
A full redesign of the WKRLY Group marketing homepage — a product studio that builds, launches, and operates its own SaaS products. This is the **"Deep Navy"** direction: a dark, structured, editorial layout that keeps the original brand's navy + teal palette but replaces the generic template look with a confident, type-led composition.

This package contains everything needed to implement the homepage: a working HTML reference, this full specification, and an agent prompt (`PROMPT.md`).

## About the Design Files
The files in this bundle are **design references created in HTML** — a prototype showing the intended look and behavior. They are **not production code to copy verbatim**. Your task is to **recreate this design in the target codebase's existing environment** (React, Vue, Svelte, Astro, plain HTML/CSS, etc.) using its established conventions, component library, and styling approach. If no codebase exists yet, choose an appropriate modern framework (e.g. Next.js + Tailwind, or Astro) and implement it there.

- `WKRLY Homepage.dc.html` — the design reference. It is a self-contained HTML file; open it in a browser to see the target. (It uses a small runtime, `support.js`, only so it renders here — you do **not** need that runtime in production; treat the rendered output as the spec.)
- `screenshot.png` — *(included only if requested)* a static capture of the full page.

## Fidelity
**High-fidelity.** Colors, typography, spacing, and layout are final and should be matched closely. Recreate the UI faithfully using the codebase's existing primitives where they exist; otherwise build to the tokens below.

## Page Structure (top → bottom)
The page is a single column, max content width **1240px**, centered. Full-bleed background bands alternate between two navy shades to separate sections. Section horizontal padding is **56px**; vertical padding is **~90px** (hero/CTA larger — see each section).

1. **Top Nav** — logo left, links + CTA right
2. **Hero** — centered headline, subhead, two buttons, soft teal radial glow
3. **Value Props** — 4-column row with hairline dividers (no card backgrounds)
4. **The Thesis** — darker band, eyebrow + large statement
5. **What We Do** — heading + "Learn how we work →" link, then a 3-card grid (1px hairline gaps)
6. **The Throughline** — darker band, quote-style statement + supporting paragraph, corner glow
7. **Our Approach** — heading + 2×2 numbered step grid
8. **Why WKRLY** — darker band, heading + 2×2 list with teal left-border accents
9. **CTA** — centered, radial glow, headline + subhead + button
10. **Footer** — darkest band, 3-column (brand blurb / Explore links / Contact & legal) + copyright row

## Interactions & Behavior
- **Nav links / buttons**: hover state — lighten text toward white (links) and slightly darken/brighten the teal fill (buttons). Add `cursor:pointer` and a ~150ms color transition. (Not shown in the static reference; apply codebase-standard hover affordances.)
- **Hero glow**: gentle opacity breathing, 8s loop. Gate behind `@media (prefers-reduced-motion: reduce)` → disable animation.
- **Buttons**: currently non-functional placeholders. Wire "Get in Touch" to the contact route/section; "Explore What We Do" and "See the Portfolio" to their respective pages/anchors.
- **Responsive** (not in reference — implement to codebase breakpoints):
  - ≤1024px: value props 4→2 cols; What-We-Do 3→1 or 2 cols; Approach/Why 2→1 col; reduce H1 to ~56px.
  - ≤640px: single column throughout; nav collapses to a menu button; H1 ~40px, section H2 ~32px; padding ~32px horizontal.

**Typography**
- Display / headings: **Sora** (Google Fonts), weights 600 / 700 / 800.
- Body / UI: **Manrope** (Google Fonts), weights 400 / 500 / 600 / 700.
- Scale: H1 80px/800, section H2 48px/700, CTA H2 54px/700, statement 40–42px/600, card title 20px/600, body 14–20px/400–500, eyebrow 12px/600 `letter-spacing:.16em` uppercase.

**Spacing & shape**
- Max content width: 1240px. Section H-padding: 56px. Section V-padding: 90px (hero 84/92, thesis 96, CTA 104).
- Radii: buttons 8–9px, icon tiles 8px, card grids 14px.
- Hairline gaps in grids: `gap:1px` over a `rgba(255,255,255,.07)` backing.
- No drop shadows on the page itself (the reference's outer shadow is just the preview frame).
